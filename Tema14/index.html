Tema 14 - Almacenamiento Web

Ejercicio 1 - Preferencias de usuario con localStorage

Permite al usuario introducir su nombre y elegir un color de fondo. Los datos se guardan con localStorage y se mantienen al recargar la página.

Abrir Ejercicio 1

Ejercicio 2 - Contador de visitas con localStorage

Implementa un contador que se incrementa automáticamente cada vez que se recarga la página. Muestra el número de visitas y permite reiniciar el contador.

Abrir Ejercicio 2

Ejercicio 3 - Almacenamiento temporal con sessionStorage

Formulario con campos nombre y curso. Los datos se guardan en sessionStorage y desaparecen al cerrar la pestaña o el navegador.

Abrir Ejercicio 3

Ejercicio 4 - Comparativa localStorage vs sessionStorage

Permite guardar el mismo dato en ambos sistemas y comparar su comportamiento al recargar la página, cerrar la pestaña o abrir una nueva.

Abrir Ejercicio 4

Ejercicio 5 - Lista de tareas con localStorage

Aplicación de lista de tareas que permite añadir, mostrar y eliminar tareas. Los datos se almacenan con localStorage y se sincronizan automáticamente entre pestañas abiertas.

Abrir Ejercicio 5
